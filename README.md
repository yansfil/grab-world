# grab-world
그랩의 실험실

## Description 
현재 어플리케이션 구성을 기획중입니다.
  
- 사용 언어 : typescript
- 서버 구성 : node, graphql, apollo-server, mysql
- 클라이언트 구성 : react, apollo-client, graphql-codegen
- 테스트 도구 : jest
- CI/CD : github actions
- 인프라(예정) : eks, lambda, kafka

## Install
```sh
yarn bootstrap
```

## Run all projects 
```sh
yarn dev
```

## Test
```sh
yarn test
```
